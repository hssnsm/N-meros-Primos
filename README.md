# Gerador de Números Primos

## Como fiz o script:

**Fiz o código em C, fui ajustando ele conforme os testes;**

**Usei um crivo segmentado para manter a performance dentro do limite de tempo;**

**Para rodar, basta compilar e executar o programa.**

# Como executar:

## Clone o repositório

```bash
git clone https://github.com/hssnext/N-meros-Primos.git
```

## Compile

```bash
gcc -O3 -march=native -pthread primos.c -o primes
```

## Execute

```bash
./primes
```
